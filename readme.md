## Usage

Clone this repository:

In `run.rb` , set the files names :

```
data_file = "my_csv_file.csv"

```

In terminal, run:

```
gem install httparty
gem install threads
gem install nokogiri
mkdir emails
```

In `run.rb` , write the interval you want to search. From 0 to 192.000
```
start_index = 0
end_index = 500

```

And run :

```
ruby run.rb
```
